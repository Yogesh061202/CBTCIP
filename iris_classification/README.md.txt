# Iris Flower Classification

## Project Overview
This project implements a machine learning model to classify iris flowers into their respective species based on their sepal and petal measurements. The Iris dataset is a classic dataset for classification, machine learning, and data visualization.

## Dataset
The Iris dataset contains 150 samples of iris flowers, with 50 samples each of three different species:
- Iris setosa
- Iris virginica
- Iris versicolor

Each sample has four features:
1. Sepal length (cm)
2. Sepal width (cm)
3. Petal length (cm)
4. Petal width (cm)
